資料の第2.2節「多値変数」の内容に基づき、二値変数（ベルヌーイ分布）の拡張として、複数の状態を取る変数の扱いを数式と図に沿って解説します。

### 1. 多値変数の表現：1-of-K 符号化法

二値変数は2つの状態（0か1）のみを扱いましたが、ここでは相互に排他的な $K$ 個の状態のうちの1つを取る離散変数を考えます。これを表現するために**1-of-K 符号化法**が用いられます。

- **表現:** $K$ 次元のベクトル $\mathbf{x}$ で表し、ある特定の状態 $k$ にあるとき、その要素 $x_k$ だけが $1$ で、残りの要素はすべて $0$ となります。
- **例:** $K=6$ の状態で、3番目の状態が選ばれた場合、ベクトルは以下のようになります（式2.25）。 $$\mathbf{x} = (0, 0, 1, 0, 0, 0)^T$$
- **制約:** 常に $\sum_{k=1}^K x_k = 1$ を満たします。

このとき、$x_k = 1$ となる確率をパラメータ $\mu_k$ とすると、$\mathbf{x}$ の分布は以下のように書けます（式2.26）。 $$p(\mathbf{x}|\boldsymbol{\mu}) = \prod_{k=1}^K \mu_k^{x_k}$$ ここで、$\mu_k \ge 0$ かつ $\sum_k \mu_k = 1$ という制約があります。

### 2. 最尤推定によるパラメータ決定

$N$ 個の独立な観測値データ $D = {\mathbf{x}_1, \dots, \mathbf{x}_N}$ が与えられたとき、尤度関数は次のように表されます（式2.29）。 $$p(D|\boldsymbol{\mu}) = \prod_{n=1}^N \prod_{k=1}^K \mu_k^{x_{nk}} = \prod_{k=1}^K \mu_k^{\left(\sum_n x_{nk}\right)} = \prod_{k=1}^K \mu_k^{m_k}$$ ここで、$m_k = \sum_n x_{nk}$ は「状態 $k$ が観測された回数」であり、この分布の**十分統計量**と呼ばれます。

$\sum_k \mu_k = 1$ という制約の下でこの尤度関数を最大化するために、**ラグランジュ乗数 $\lambda$** を導入します（式2.31）。 $$\sum_k m_k \ln \mu_k + \lambda \left(\sum_k \mu_k - 1\right)$$ これを $\mu_k$ で微分して $0$ と置くと、最尤推定値として直感的に納得できる「観測された割合」が得られます（式2.33）。 $$\mu_k^{ML} = \frac{m_k}{N}$$

### 3. 多項分布 (Multinomial distribution)

単一の試行ではなく、$N$ 回の試行で各状態 $k$ が $m_k$ 回ずつ現れる**同時確率**を考えたものが**多項分布**です（式2.34）。 $$\text{Mult}(m_1, m_2, \dots, m_K | N, \boldsymbol{\mu}) = \binom{N}{m_1 m_2 \dots m_K} \prod_{k=1}^K \mu_k^{m_k}$$ 先頭の項は、合計 $N$ 個の対象を、各グループの大きさが $m_1, \dots, m_K$ となる $K$ 個のグループに分割する組み合わせの数（多項係数）です。

### 4. ディリクレ分布：ベイズ的な事前分布

二値変数のときにベータ分布を導入したのと同様に、多値変数のパラメータ $\boldsymbol{\mu}$ に対してもベイズ的な扱いをするため、事前分布として**ディリクレ分布 (Dirichlet distribution)** を導入します。

- **定義 (式2.38):** $$\text{Dir}(\boldsymbol{\mu}|\boldsymbol{\alpha}) = \frac{\Gamma(\alpha_0)}{\Gamma(\alpha_1) \dots \Gamma(\alpha_K)} \prod_{k=1}^K \mu_k^{\alpha_k-1}$$ ここで $\alpha_k$ は超パラメータであり、$\alpha_0 = \sum \alpha_k$ です。

#### 図 2.4 と 図 2.5 の解説

- **図 2.4:** パラメータ $\mu_k$ は $\sum \mu_k = 1$ という制約があるため、その分布の定義域は $K-1$ 次元の**単体 (simplex)** の上に制限されます。例えば $K=3$ の場合、3次元空間内の正三角形の面上に分布が存在することを視覚的に示しています。
- **図 2.5:** 超パラメータ $\alpha_k$ の値を変えたときの分布の様子です。
    - $\alpha_k$ が小さい（例：0.1）と、分布は単体の角（各状態が極端に偏る場所）に集中します。
    - $\alpha_k$ を大きくする（例：10）と、分布は中心付近に鋭く集まり、「どの状態も平均的に起こる」という確信が強まります。

### 5. 事後分布の更新

事前分布であるディリクレ分布に尤度関数（多項分布）を掛けると、事後分布もまたディリクレ分布になります。 $$p(\boldsymbol{\mu}|D, \boldsymbol{\alpha}) \propto p(D|\boldsymbol{\mu}) p(\boldsymbol{\mu}|\boldsymbol{\alpha}) \propto \prod_{k=1}^K \mu_k^{\alpha_k + m_k - 1}$$ これより、事後分布のパラメータは単に **$\alpha_k + m_k$** となり、事前分布のパラメータ $\alpha_k$ が「各状態の仮想的な観測回数」として機能していることがわかります。

このように、二値変数における「ベルヌーイ分布・ベータ分布」の関係が、多値変数では「多項分布・ディリクレ分布」へと、より一般的な形に拡張されています。
