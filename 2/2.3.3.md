第2.3.3節「ガウス変数に対するベイズの定理」では、あるガウス分布 $p(\mathbf{x})$ と、それに基づいて定義される条件付きガウス分布 $p(\mathbf{y}|\mathbf{x})$ が与えられたときに、**逆の条件付き分布 $p(\mathbf{x}|\mathbf{y})$ や周辺分布 $p(\mathbf{y})$ を導出する**手順が解説されています。

この節の内容は、後に詳述される「線形ガウスモデル」の基礎となる非常に重要な議論です。

---

### 1. 設定：事前分布と条件付き分布 (式 2.99, 2.100)

まず、以下の2つのガウス分布が与えられていると仮定します。

- **$\mathbf{x}$ の周辺分布（事前分布）:** 平均 $\boldsymbol{\mu}$、精度行列 $\mathbf{\Lambda}$ のガウス分布です。 $$p(\mathbf{x}) = \mathcal{N}(\mathbf{x} | \boldsymbol{\mu}, \mathbf{\Lambda}^{-1}) \quad (2.99)$$
- **$\mathbf{x}$ が与えられた下の $\mathbf{y}$ の条件付き分布:** 平均が $\mathbf{x}$ の線形関数 $\mathbf{A}\mathbf{x} + \mathbf{b}$ で表され、精度行列 $\mathbf{L}$ を持つガウス分布です。 $$p(\mathbf{y}|\mathbf{x}) = \mathcal{N}(\mathbf{y} | \mathbf{A}\mathbf{x} + \mathbf{b}, \mathbf{L}^{-1}) \quad (2.100)$$

### 2. ステップ1：同時分布 $p(\mathbf{z})$ の特定 (式 2.101 - 2.108)

周辺分布 $p(\mathbf{y})$ などを求めるために、まず $\mathbf{x}$ と $\mathbf{y}$ を合わせたベクトル $\mathbf{z} = \begin{pmatrix} \mathbf{x} \ \mathbf{y} \end{pmatrix}$ の同時分布を求めます。

- **対数尤度の和 (式 2.102):** ベイズの定理の構成要素を対数で考えると、$\ln p(\mathbf{z}) = \ln p(\mathbf{x}) + \ln p(\mathbf{y}|\mathbf{x})$ となります。
- **精度行列 $\mathbf{R}$ の導出 (式 2.104):** 指数部にある $\mathbf{x}$ と $\mathbf{y}$ の2次形式の項を整理すると、同時分布の精度行列 $\mathbf{R}$ が以下のように求まります。 $$\mathbf{R} = \begin{pmatrix} \mathbf{\Lambda} + \mathbf{A}^T\mathbf{L}\mathbf{A} & -\mathbf{A}^T\mathbf{L} \\ -\mathbf{L}\mathbf{A} & \mathbf{L} \end{pmatrix} \quad (2.104)$$
- **同時分布の平均 (式 2.108):** 同様に、$\mathbf{x}$ と $\mathbf{y}$ の1次の項を調べることで、同時分布の平均 $\mathbb{E}[\mathbf{z}]$ が得られます。 $$\mathbb{E}[\mathbf{z}] = \begin{pmatrix} \boldsymbol{\mu} \ \mathbf{A}\boldsymbol{\mu} + \mathbf{b} \end{pmatrix} \quad (2.108)$$

### 3. ステップ2：$\mathbf{y}$ の周辺分布 (式 2.109, 2.110)

同時分布が求まれば、第2.3.2節で学んだ「周辺化」の公式を適用することで、$\mathbf{y}$ だけの分布を抽出できます。

- **平均 (式 2.109):** $\mathbb{E}[\mathbf{y}] = \mathbf{A}\boldsymbol{\mu} + \mathbf{b}$
- **共分散 (式 2.110):** $\text{cov}[\mathbf{y}] = \mathbf{L}^{-1} + \mathbf{A}\mathbf{\Lambda}^{-1}\mathbf{A}^T$
    - これは、**「元の $\mathbf{y}$ のノイズ ($\mathbf{L}^{-1}$)」に「$\mathbf{x}$ の不確実性が $\mathbf{A}$ を通じて伝播した分 ($\mathbf{A}\mathbf{\Lambda}^{-1}\mathbf{A}^T$)」が足し合わされたもの**と解釈できます。

### 4. ステップ3：条件付き分布 $p(\mathbf{x}|\mathbf{y})$ (式 2.111, 2.112)

最後に、観測値 $\mathbf{y}$ が得られた後の $\mathbf{x}$ の分布を求めます。これは第2.3.1節の「条件付き分布」の公式を同時分布 $p(\mathbf{z})$ に適用することで得られます。

- **平均 (式 2.111):** $\mathbb{E}[\mathbf{x}|\mathbf{y}] = (\mathbf{\Lambda} + \mathbf{A}^T\mathbf{L}\mathbf{A})^{-1} { \mathbf{A}^T\mathbf{L}(\mathbf{y} - \mathbf{b}) + \mathbf{\Lambda}\boldsymbol{\mu} }$
- **精度行列 (式 2.112):** $\text{cov}[\mathbf{x}|\mathbf{y}]^{-1} = \mathbf{\Lambda} + \mathbf{A}^T\mathbf{L}\mathbf{A}$
    - この精度行列は、**「事前知識の精度 ($\mathbf{\Lambda}$)」に「データ $\mathbf{y}$ から得られた精度 ($\mathbf{A}^T\mathbf{L}\mathbf{A}$)」を足したもの**になっています。

---

### まとめ：分割されたガウス分布の公式

資料の最後には、これらの結果が「ガウス分布の周辺分布と条件付き分布」として美しくまとめられています（式 2.113 - 2.117）。

この一連の流れを視覚的に理解する際は、**図 2.9** を思い出すと助けになります。同時分布 $p(\mathbf{x}, \mathbf{y})$ という2次元の「山」があるとき、周辺分布 $p(\mathbf{y})$ はその山を片方の軸に押しつぶした影であり、条件付き分布 $p(\mathbf{x}|\mathbf{y})$ は特定の $\mathbf{y}$ で山をスパスパと切り出した断面に対応しています。

この節で導出された公式は、**カルマンフィルタなどの時系列モデルや、線形回帰のベイズ推定**において、観測データに基づいて未知のパラメータを更新する際の数学的基盤となっています。
