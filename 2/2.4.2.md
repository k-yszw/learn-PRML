資料の第2.4.2節「共役事前分布」の内容について、数式に基づき、話の流れに沿って解説します。

これまで、ベルヌーイ分布に対するベータ分布や、多項分布に対するディリクレ分布など、個別の分布について**共役事前分布**を見てきました。本節では、第2.4節で定義された「指数型分布族」という一般的な枠組みにおいて、**どのような形式の事前分布を選べば共役性が成り立つのか**を数学的に導き出しています。

### 1. 尤度関数の再確認 (式 2.227)

まず、指数型分布族に属する分布から得られた $N$ 個のデータ集合 $\mathbf{X}$ に対する尤度関数は、前節で導出した通り以下の形になります。

- **式 (2.227):** $$p(\mathbf{X}|\boldsymbol{\eta}) = \left( \prod_{n=1}^N h(\mathbf{x}_n) \right) g(\boldsymbol{\eta})^N \exp \left\{ \boldsymbol{\eta}^T \sum_{n=1}^N \mathbf{u}(\mathbf{x}_n) \right\}$$ この式において、パラメータ $\boldsymbol{\eta}$ に依存している部分は **$g(\boldsymbol{\eta})^N$** と **$\exp{\boldsymbol{\eta}^T \dots }$** の箇所です。

### 2. 指数型分布族に対する共役事前分布の定義 (式 2.229)

**共役性 (conjugacy)** とは、事前分布に尤度関数を掛けた結果（事後分布）が、元の事前分布と同じ関数形式になる性質を指します。そのため、事前分布 $p(\boldsymbol{\eta})$ は尤度関数の $\boldsymbol{\eta}$ に依存する部分と同じ形をしている必要があります。

資料では、指数型分布族の任意の分布に対して、以下の形式の事前分布を定義しています。

- **式 (2.229):** $$p(\boldsymbol{\eta}|\boldsymbol{\chi}, \nu) = f(\boldsymbol{\chi}, \nu) g(\boldsymbol{\eta})^\nu \exp { \nu \boldsymbol{\eta}^T \boldsymbol{\chi} }$$ ここで、$\boldsymbol{\chi}$ と $\nu$ は事前分布の形状を制御する**超パラメータ**であり、$f(\boldsymbol{\chi}, \nu)$ は分布を規格化するための係数です。

### 3. 事後分布の導出と更新ルール (式 2.230)

実際に、この事前分布 (2.229) に尤度関数 (2.227) を掛けて事後分布を求めると、非常にシンプルな更新式が得られます。

- **式 (2.230):** $$p(\boldsymbol{\eta}|\mathbf{X}, \boldsymbol{\chi}, \nu) \propto g(\boldsymbol{\eta})^{\nu+N} \exp \left\{ \boldsymbol{\eta}^T \left( \sum_{n=1}^N \mathbf{u}(\mathbf{x}_n) + \nu \boldsymbol{\chi} \right) \right\}$$ この結果を見ると、事後分布もまた式 (2.229) と同じ形式の分布になっていることが分かります。これにより、指数型分布族という広いクラスに対して、**共役事前分布が常に存在すること**が証明されました。

### 4. パラメータの解釈：仮想的な観測値

この更新式から、超パラメータ $\nu$ と $\boldsymbol{\chi}$ には直感的な意味があることが分かります。

- **$\nu$:** 事前分布における **「有効な観測値の数」** として解釈できます。事後分布では、実際のデータ数 $N$ がこれに加算されます。
- **$\boldsymbol{\chi}$:** 観測された十分統計量 $\mathbf{u}(\mathbf{x})$ に対応する、**「仮想的な観測値の平均」** のような役割を果たします。

### まとめ

この節の議論により、個別の分布ごとに共役事前分布を考案しなくても、**「指数型分布族の一般形式に合わせて事前分布を構成すれば、自動的に共役性が得られる」** という強力な結論が導かれました。

例えば、ベルヌーイ分布において $\nu$ と $\boldsymbol{\chi}$ を適切に設定すればベータ分布の更新式（式 2.18）と一致し、多変量ガウス分布であれば精度行列に対するウィシャート分布（式 2.155）などの具体的な関係も、この一般論の一部として統合的に理解することができます。
