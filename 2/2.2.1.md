資料の第2.2.1節に基づき、多値変数のためのベイズ的な事前分布である**ディリクレ分布**について、数式と図を交えて流れに沿って解説します。

### 1. 導入：多項分布の「共役事前分布」

前節（2.2節）では、サイコロのように $K$ 個の状態がある多値変数を扱いました。そのパラメータ $\boldsymbol{\mu} = (\mu_1, \dots, \mu_K)^T$ をベイズ的に扱うためには、適切な事前分布が必要です。 多項分布の尤度関数の形（$\prod \mu_k^{m_k}$）を見ると、事前分布として $\mu_k$ のべき乗の形をした分布を選べば、事後分布も同じ形になり計算が非常に楽になります。この性質を「共役性」と呼び、選ばれるのが**ディリクレ分布**です。

### 2. ディリクレ分布の定義（式 2.38）

ディリクレ分布は、パラメータ $\boldsymbol{\mu}$ の確からしさを表す分布で、次のように定義されます。

- **式 (2.38):** $$\text{Dir}(\boldsymbol{\mu}|\boldsymbol{\alpha}) = \frac{\Gamma(\alpha_0)}{\Gamma(\alpha_1) \dots \Gamma(\alpha_K)} \prod_{k=1}^K \mu_k^{\alpha_k-1}$$
- **パラメータ $\alpha_k$:** 超パラメータと呼ばれ、$\boldsymbol{\alpha} = (\alpha_1, \dots, \alpha_K)^T$ で表されます。
- **規格化係数:** 分布を積分して1にするための係数で、ガンマ関数 $\Gamma$ を用いて計算されます。ここで $\alpha_0 = \sum_{k=1}^K \alpha_k$ です（式 2.39）。

### 3. 図 2.4：分布が存在する「舞台（単体）」

ディリクレ分布を理解する上で重要な制約が、$\sum \mu_k = 1$ かつ $0 \le \mu_k \le 1$ であることです。

- **図 2.4:** $K=3$ の場合、3つのパラメータ $\mu_1, \mu_2, \mu_3$ は自由勝手な値を取れるわけではなく、3次元空間内の **「正三角形の面上（単体; simplex）」** に制限されます。ディリクレ分布はこの平面の上で、どこが「高密度（もっともらしいか）」を表現します。

### 4. 図 2.5：超パラメータ $\alpha$ による形の変化

資料の**図 2.5**は、$\alpha_k$ の値を変えたときに、単体上の分布がどう変化するかを示しています。

- **$\alpha_k = 0.1$（左）:** 分布が三角形の頂点付近に集中します。これは「どれか一つの状態が極端に強く起こる」という予測を表します。
- **$\alpha_k = 1$（中央）:** 分布が平らになります。どの $\mu$ の組み合わせも同じくらいあり得ると考えている状態です。
- **$\alpha_k = 10$（右）:** 分布が中央に鋭く集まります。これは「どの状態も平均的に起こるはずだ」という確信が非常に強い状態です。

### 5. 事後分布の更新：学習の仕組み

実際にデータを観測し、各状態 $k$ が $m_k$ 回ずつ現れたとします。このとき、事前分布（ディリクレ分布）に尤度（多項分布）を掛けて事後分布を求めると、その結果は再びディリクレ分布になります。

- **更新後のパラメータ:** 新しい分布のパラメータは、単に **$\alpha_k + m_k$** となります。

このことから、事前分布のパラメータ $\alpha_k$ は、データを見る前の **「仮想的な観測回数」** として解釈できることが分かります。

### まとめ

ディリクレ分布は、二値変数のときに学んだ「ベータ分布」を多次元へと拡張したものです。**「現在の予想（$\alpha_k$）」に「実際の観測（$m_k$）」を足すだけ**で、多値変数のパラメータに関する知識を更新できる、非常に効率的でベイズ的な仕組みを提供しています。
