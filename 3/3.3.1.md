第3.3.1節「パラメータの分布」では、線形回帰モデルのパラメータ $\mathbf{w}$ に対してベイズ的な視点を導入し、データを通じてその「不確実性」がどのように解消されていくかを数理的・視覚的に解説しています。

話の流れに沿って、数式の意味と図の解釈を詳しく説明します。

### 1. ベイズ的アプローチの出発点：事前分布の設定

これまでの最尤推定では、パラメータ $\mathbf{w}$ は「たった一つの最適な値」として求められてきました。ベイズ的な線形回帰では、$\mathbf{w}$ 自体がある確率分布に従う変数であると考え、観測データを得る前の信念を**事前分布 $p(\mathbf{w})$** として導入します。

資料では、解析的な扱いやすさ（共役性）を考慮して、$\mathbf{w}$ が平均 $\mathbf{m}_0$、共分散行列 $\mathbf{S}_0$ の**ガウス事前分布**に従うと仮定します。 $$p(\mathbf{w}) = \mathcal{N}(\mathbf{w}|\mathbf{m}_0, \mathbf{S}_0) \quad (3.48)$$

### 2. 事後分布の導出：不確実性の更新

データ集合 $\mathbf{t}$ を観測した後のパラメータの分布である**事後分布 $p(\mathbf{w}|\mathbf{t})$** を求めます。ベイズの定理により、これは「事前分布 × 尤度関数」に比例します。

- **数理的導出のポイント:** 第2.3.1節で学んだ「ガウス分布の指数部分の比較（平方完成）」という手法がここでも使われます。事前分布も尤度関数も $\mathbf{w}$ の2次形式の指数関数であるため、それらを掛け合わせた事後分布もまた**ガウス分布**になります。
- **事後分布のパラメータ (式 3.50, 3.51):** 事後平均 $\mathbf{m}_N$ と事後共分散の逆行列（精度行列） $\mathbf{S}_N^{-1}$ は次のように更新されます。 $$\mathbf{S}_N^{-1} = \mathbf{S}_0^{-1} + \beta \boldsymbol{\Phi}^T \boldsymbol{\Phi} \quad (3.51)$$ $$\mathbf{m}_N = \mathbf{S}_N (\mathbf{S}_0^{-1} \mathbf{m}_0 + \beta \boldsymbol{\Phi}^T \mathbf{t}) \quad (3.50)$$

**数式の直感的な意味:** 事後の精度（確信度）は、事前の精度に「データの精度」が積み上がることで高まります（式 3.51）。また、事後平均は、事前の平均とデータからの推定値をそれぞれの精度で重み付けした平均値として計算されます。

### 3. 特別なケース：正則化とのつながり

事前分布として、平均 $0$、共分散 $\alpha^{-1}\mathbf{I}$ の単純な等方性ガウス分布（式 3.52）を仮定した場合、事後分布を最大化すること（MAP推定）は、第3.1.4節で学んだ**正則化（荷重減衰）付きの最小二乗法**と数学的に等価になります。 このとき、正則化係数は **$\lambda = \alpha / \beta$** となり、事前知識の強さ $\alpha$ とノイズの精度 $\beta$ の比率によって正則化の強さが決まることが示されています。

### 4. 図3.7の解釈：逐次ベイズ学習の可視化

この節の最も重要な視覚的資料が **図3.7** です。ここでは、シンプルな直線モデル $y(x, \mathbf{w}) = w_0 + w_1 x$ を例に、データが1点ずつ追加されるたびにパラメータ空間の分布がどう変わるかが描かれています。

- **1行目（データなし）:**
    - **中央（事前分布）:** パラメータ空間の中心（$0, 0$）をピークとする円形の分布です。まだ何の知識もありません。
    - **右（データ空間）:** この事前分布からランダムに $\mathbf{w}$ をサンプリングして描いた直線です。あらゆる方向を向いています。
- **2行目（データ1点目）:**
    - **左（尤度）:** 1点を通る直線は無限にありますが、その傾きと切片の関係は「帯状」に制限されます。
    - **中央（事後分布）:** 事前分布と尤度が掛け合わされ、分布が斜めに引き伸ばされます。
    - **右（データ空間）:** 直線群は、およそ観測された青い点を通るようになります。
- **3行目（データ2点目）:**
    - **左（尤度）:** 2点目に関する別の方向の帯状の尤度です。
    - **中央（事後分布）:** 1点目の事後分布（今や事前分布）と、新しい尤度が交差する位置に分布が凝縮されます。これにより、パラメータの値がかなり絞り込まれます。
- **4行目（データ20点目）:**
    - **中央（事後分布）:** 非常に鋭いピーク（十字の真の値を指す）になります。不確実性がほぼ解消された状態です。
    - **右（データ空間）:** すべての直線が重なり、真の直線を精度よく再現しています。

### まとめ

第3.3.1節は、ベイズ推論の**「以前の事後分布を次の事前分布として再利用する」**という逐次的な学習プロセスを数理と図の両面で示しています。 これにより、頻度主義のように「多くのデータセット」を仮定しなくても、**「単一のデータセット内で、観測が増えるごとにパラメータの不確実性（分布の広がり）が減少していく様子」**を自然に記述できることが、この節の核心的なメッセージです。
