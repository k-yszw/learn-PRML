第3.3.3節「等価カーネル」の内容について、ベイズ線形回帰の予測が「既存のデータの重み付き平均」として解釈できるという非常に興味深い視点を、数式と図を用いて解説します。

### 1. 数理的な導出：予測平均の書き換え

前節（第3.3.2節）で、新しい入力 $\mathbf{x}$ に対する予測分布の平均は、事後平均 $\mathbf{m}_N$ を用いて $y(\mathbf{x}, \mathbf{m}_N) = \mathbf{m}_N^T \boldsymbol{\phi}(\mathbf{x})$ となることを見ました。この式を、訓練データの目標値ベクトル $\mathbf{t} = (t_1, \dots, t_N)^T$ が直接見える形に書き換えてみます。

- **代入操作:** 事後平均の定義式 $\mathbf{m}_N = \beta \mathbf{S}_N \boldsymbol{\Phi}^T \mathbf{t}$ （式 3.53より、$\mathbf{m}_0 = \mathbf{0}$ の場合）を代入します。 $$y(\mathbf{x}, \mathbf{m}_N) = \boldsymbol{\phi}(\mathbf{x})^T \beta \mathbf{S}_N \boldsymbol{\Phi}^T \mathbf{t} = \sum_{n=1}^N \beta \boldsymbol{\phi}(\mathbf{x})^T \mathbf{S}_N \boldsymbol{\phi}(\mathbf{x}_n) t_n \quad (3.60)$$
- **等価カーネルの定義:** ここで、新しい入力 $\mathbf{x}$ と訓練データ $\mathbf{x}_n$ の関係を決める関数を **$k(\mathbf{x}, \mathbf{x}_n)$** と置くと、予測平均は次のシンプルな形で表現できます。 $$y(\mathbf{x}, \mathbf{m}_N) = \sum_{n=1}^N k(\mathbf{x}, \mathbf{x}_n) t_n \quad (3.61)$$ この関数 $k(\mathbf{x}, \mathbf{x}')$ を**等価カーネル（equivalent kernel）**と呼びます。

### 2. 数式の意味：線形平滑器（Linear Smoother）

この式(3.61)は、**「新しい地点の予測値は、訓練データの目標値 $t_n$ をカーネル関数 $k(\mathbf{x}, \mathbf{x}_n)$ によって重み付けして足し合わせたものである」**という直感的な事実を示しています。

- **重みの役割:** $k(\mathbf{x}, \mathbf{x}_n)$ は、予測したい地点 $\mathbf{x}$ に対して、各訓練データ $x_n$ がどの程度の影響力（重み）を持つかを決定します。
- **平滑化:** 目標値 $t_n$ に重みを掛けて平均化する操作であることから、この予測プロセスは**線形平滑器（linear smoother）**とも呼ばれます。

### 3. 図の解釈：カーネルの形状と局所性

資料の **図3.10** と **図3.11** は、異なる基底関数から導かれた等価カーネルの形状を可視化しています。

- **図3.10（ガウス基底）:** ガウス基底関数から導かれた等価カーネル $k(x, x')$ が描かれています。
    - **解釈:** カーネルは $x$ の周りに鋭いピークを持ち、**局所的（localized）**な性質を持っていることがわかります。つまり、予測したい地点 $x$ に近いデータほど大きな重みが与えられ、遠いデータの影響は急速に小さくなります。
- **図3.11（多項式・シグモイド基底）:** 驚くべきことに、多項式基底のような「大域的」な基底関数（空間全体に影響が及ぶもの）から導かれた等価カーネルであっても、**図3.11** に見られるように、やはり $x$ の近傍で大きな値を持ち、遠くで減衰する局所的な形状を示します。

### 4. 等価カーネルの重要な性質

資料では、等価カーネルが持ついくつかの数学的特性が紹介されています。

1. **和が1になる性質:** 任意の $\mathbf{x}$ に対して、全訓練データにわたる重みの合計は $1$ になります。 $$\sum_{n=1}^N k(\mathbf{x}, \mathbf{x}_n) = 1 \quad (3.64)$$ これは、予測がデータの単なる加重平均であることを保証する、平滑化器として健全な性質です。
2. **分散との関係:** 予測分布における2点 $y(\mathbf{x})$ と $y(\mathbf{x}')$ の間の共分散は、等価カーネルに比例します。 $$cov[y(\mathbf{x}), y(\mathbf{x}')] = \beta^{-1} k(\mathbf{x}, \mathbf{x}')$$ これは、カーネルの値が大きい（＝2点が近い）ほど、その地点での予測値同士が強く相関することを意味します。

### 5. まとめ：カーネル法への橋渡し

第3.3.3節の議論は、これまでの「基底関数を設計する」というアプローチを、**「カーネル関数を直接設計する」**という全く新しいアプローチへと昇華させます。

- **話の流れ:** これまでは基底関数 $\boldsymbol{\phi}(\mathbf{x})$ を固定して重み $\mathbf{w}$ を学習してきましたが、等価カーネルの視点に立てば、**基底関数を介さずに、データ間の類似度（カーネル）を直接定義して予測を行うことが可能**になります。
- **次なるステップ:** このアイデアをさらに発展させたものが、資料内でも言及されている**ガウス過程（Gaussian process）**です。第3章の線形モデルの議論は、ここで「カーネル」というキーワードを介して、より高度な非パラメトリックモデル（第6章）へと繋がっていくことになります。
